# SlateMode Component

## Purpose
The SlateMode component provides the Alchemist's Slate Mode interface, allowing players to design and manage alchemical machines by connecting nodes in a visual programming environment.

## Features
- Displays the Alchemist's Slate with a node-based visual programming interface
- Shows the Infinity Well sidebar for viewing resources and items
- Provides a full-featured slate editor for designing alchemical machines
- Currently includes example nodes for demonstration

## Usage
This component is one of the main game modes that players can switch to from the main menu.
