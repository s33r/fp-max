# Counter

A React component that displays a counter with increment and decrement functionality.

## Features

- Uses Decimal.js for precise numerical calculations
- Validates props using Zod schema
- Configurable initial value and step size
- Styled with gradient background and smooth animations

## Props

- `initialValue` (number, default: 0): The starting count value
- `step` (number, default: 1): The increment/decrement amount per button click
