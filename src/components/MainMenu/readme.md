# MainMenu Component

## Purpose
The MainMenu component provides the top-level navigation menu that allows players to switch between the five main game modes.

## Features
- Displays navigation buttons for all five game modes: Alchemist's Slate, Status, Conversation, Library, and Menu
- Highlights the currently active mode
- Provides icons and labels for each mode
- Follows the game's visual design with dark theme styling

## Usage
This component is displayed at the top of the App and manages mode switching throughout the game.
